JavaScript & HTML5 Comic Reader

Simply drag and drop a CBZ or CBR file onto the page to start reading.

Started as a weekend project by Raymond Camden. eComic Reader is built upon both Robert Kuykendall's Web Slinger and Jeff Schiller's Kthoom. Feel free to check out the code on GitHub, and comment or build upon it yourself!
